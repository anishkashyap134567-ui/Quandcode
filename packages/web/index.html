Core Subsystems

Engineered for security, speed, and local usability.

🔌

75+ LLM Providers

Unified abstraction layer supporting Anthropic, OpenAI, Gemini, and local **Ollama** models. Includes local credentials auto-fallback and automatic IP address protocol resolving.

🛡️

Rust-powered Sandbox

Executes shell/bash commands and file operations within an OS-level sandbox (Windows Job Objects & Unix namespaces) keeping your workspace secure.

💻

Cyberpunk Terminal UI

Interactive REPL built with terminal layouts, featuring realtime output tokens count, duration trackers, cost estimations, and full session resume memory.

Get Started Local

Install, configure, and launch the terminal UI in seconds.

Terminal 
# Clone and enter the repository
git clone https://github.com/anishkashyap134567-ui/Quandcode.git
cd Quandcode

# Install workspaces and runtime dependencies
bun install

# Run the interactive TUI
bun run quandcode
